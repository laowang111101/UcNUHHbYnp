# Python图书信息管理系统 python329

## 项目概述

本项目是一款基于Python语言的图书信息管理系统，包括源码、开发文档、运行教程以及开发环境。系统采用Django框架进行开发，前端使用HTML、CSS和JavaScript技术。

## 技术栈

- 后端：Python（Django框架）
- 数据库：MySQL
- 前端：HTML、CSS、JavaScript

## 功能模块

- **用户模块**
  - 登录
  - 注册
- **书籍信息管理**
  - 书籍信息展示
  - 书籍信息编辑
  - 书籍信息删除
- **书籍排序**
  - 按照不同条件对书籍进行排序
- **书籍搜索**
  - 关键字搜索书籍
- **出版社信息管理**
  - 出版社信息展示
  - 出版社信息编辑
  - 出版社信息删除
- **作者信息管理**
  - 作者信息展示
  - 作者信息编辑
  - 作者信息删除
- **分页查询**
  - 对数据列表进行分页显示

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- Python 3.x
- Django 2.x
- MySQL 5.x
